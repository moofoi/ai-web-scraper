# AI Web Scraper & Summarizer

Enter any URL and let AI analyze, summarize, 
and extract insights from website content automatically.

## Features
- Scrape any public website
- AI-powered summarization
- Extract key points & contact info
- Analyze products/services
- Custom analysis tasks

## Tech Stack
Python • BeautifulSoup • Gemini API • Streamlit
